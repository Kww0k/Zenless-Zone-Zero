## 介绍
仿绳网写的一个项目，采用前后端分离，人不分离的方式来开发。<br>
还原的页面有：
- 首页
- 事件页
- 个人中心页
- 聊天页
### 技术栈：
#### 前端：
- Vue3
- Vue-Router
- Element-Plus
- masonry-layout
- wangedit
#### 后端：
- SpringBoot
- Spring Security
- Myabtis-Plus
- Lombok
- WebSocket
- Netty
- JWT等
#### 数据库：MySql + Redis
#### 开发工具：
- IDEA
- PostMan
- Safari
